How to make colored text in Discord
To make colored text in Discord, create an ansi code block and paste it into a supported Discord client. This tool builds the code block for you.
  1. Type or paste your Discord message.
  2. Select the word or line you want to color, or leave nothing selected to style the full message.
  3. Choose a supported ANSI foreground or background color, bold, underline, or the Rainbow preset.
  4. Copy the complete fenced ansi block.
  5. Paste the full block into a supported Discord desktop or web client.
Do not remove the triple backticks or the ansi label. Discord needs the fenced code block format to interpret ANSI styles.


Discord ANSI color codes and supported styles
Discord ANSI text uses a small set of terminal-style colors. It does not support arbitrary hex colors or full RGB chat colors. The palette below uses the ANSI 30–37 foreground and 40–47 background codes, with an approximate preview because Discord rendering can vary by client and version.       Why Discord colored text may not work
      If your Discord colored text does not render, the most common reasons are client support, missing ansi code block formatting, removed escape characters, unsupported colors, or a mobile client showing a different result.
      • Make sure the copied text starts with triple backticks followed by ansi.
      • Paste the complete block, including the closing triple backticks.
      • Do not replace the invisible escape characters manually.
      • Try Discord desktop or web if mobile does not render the colors.
      • Use the supported ANSI palette instead of custom hex colors.
